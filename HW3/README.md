# HW3 — JavaPeaks Web Directory

An interactive web directory of mountains in Java, Indonesia, with map integration, search, filtering, and WhatsApp basecamp contacts.

See the [main README](../README.md#hw3--javapeaks-web-directory) for a detailed explanation of the problem, approach, implementation, and design decisions.

## Files

| File | Purpose |
|------|---------|
| `index.html` | Main page with layout, filters, map, and modal |
| `style.css` | Custom styles complementing Tailwind CSS |
| `script.js` | Data loading, filtering, map, and modal logic |
| `gunung.json` | Mountain data (name, height, difficulty, region, paths, contacts) |

## Quick Start

Open `index.html` in any modern browser.
